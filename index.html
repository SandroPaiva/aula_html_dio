O que é HTML?

(voltar)

Criada em 1989 pelo cientista britanico Tim Berners-Lee, a Linguagem de Marcação de Hipertexto, ou Hypertext Markup Language (HTML), era usada para criar e exibir conteúdo na Web quando Berners desenvolveu a ideia da Word Wide Web.

O HTML só teve sua primeira versão realmente lançada em 1991 que, diferente de sua vesão anterior não ocifial, possuia tags basicas para formatação de texto, como H1 para títulos, P para parágrafos, I para italico e B para negrito, além de incluir imagens, mesmo que em preto e branco.

Passando por significativas mudanças em suas versões, a 2.0 de 1995 incluia suprte a tabelas, a 3.0 de 1997 já permitia o uso de CSS e elementos multimidia como video e som, a versão 4.0 de 1999 inseriu novos recuros para tabelas, introduziu o IFrame e suporte a scripts em Java Script, além de incluir especificações de XHTML que combinava o XML com HTML.

Em 2000 o W3C (World Wide Web Consortium) lançou o HTML 4.01, que era uma atualização menor do HTML 4.0, corrigindo alguns erros e inconsistências.

Em 2004, o desenvolvimento do HTML foi retomado com a criação do grupo de trabalho HTML da W3C. Esse grupo trabalhou no desenvolvimento do HTML5, que foi lançado em 2014. O HTML5 incluiu muitos novos recursos, como elementos semânticos (como ARTICLE e SECTION), suporte para vídeo e áudio mais avançados, geolocalização e capacidade de armazenamento local.

Desde então, o HTML5 se tornou a versão mais utilizada do HTML e é suportado por todos os principais navegadores da web. Além disso, o HTML5 continua a ser atualizado com novos recursos e funcionalidades para acompanhar as demandas em constante evolução da web.

Estrutura básica - As tags HTML, HEAD e BODY

(voltar)

Uma pagina web é estruturada por três tags principais:

  • < HTML> - Tag principal, indica o inicio < html> e o fim < / html> do documento;
  • < HEAD> - É o cabecãlho da página, entre sua abertura < head> e fechamento < / head> é onde são inseridos as metatags e os links de carregamento de recursos que precisam ser iniciados antes do conteúdo ser exibido.
  • < BODY> - É onde é inserido tudo que deve ser exibido pelo usuário e possui também uma abertura < body> e um fechamento < / body>

Porque usar Tags e quais os tipos?

(voltar)

Uma tag é usada para definir a formatação em um documento HTML, é a marcação usada para o navegador endender onde uma linha precisa ser quebrada, quando um texto deve estar em negrito e quantas linhas e colunas uma tabela deve possuir.

Ao usar as Tags, estou determinando como o os objetos da página irão aparecer.

Existem, basicamente dois tipos de tags:

  • Com fechamento: Elas delimitam o espaço de ação da formatação, por exemplo, em um texto em negrito, uso a tag STRONG, abrindo antes e fechando depois de uma palabra ou frase que eu desejo "negritar", então eu uso < strong> texto < /strong> , o mesmo para o itálico, ou a tag I, < i> texto < /i>

As tags por si só já permitem uma organização na página HTML, mas ainda precisam dos chamados parâmetros que, podem ser exclusivos, com no caso da tag INPUT, que possui o parêmetro TYPE que determina se será um campo de texto, de senha, de email, de telefone e outros tipos.

Mas exsitem os parâmetros que são comuns para todas as tags, como ID que determina uma itentificação da tag para facilitar na hora de trabahar com CSS, o Class que permite que seja definida uma classe CSS para a tag, STYLE que permite o uso de CSS inline.

Tag de parágrafo e quebra de linha

(voltar)

Ao escrever em um documento HTML, o navegador não entenderá se for dado enter quando quero mudar de linha. Então o HTML possui uma tag para isso, que é o BR < br> , que onde for colocado o navegador entenderá que ali termina uma linha e precisa continuar na de baixo.

OBS: O BR não precisa ser fechado

O mesmo ocorre com um parágrafo, o navegador não entende quando começa e termina um parágrafo, então é usado o < p> para iniciar um parágrafo e o < / p> para fecha-lo, isso define que cada bloco de texto com essa tag é um parágrafo.

Tag de titulo e subtitulos

(voltar)

Todo documento possui título e pelo menos um sub-título, em um documento HTML não é diferente.

A Tag usada para definir titulos e sub-titulos é o H que tem níveis que vão do 1 ao 6, onde o 1 é o maio e o 6 o menor:

  • < h1> ... < / h1>
  • < h2> ... < / h2>
  • < h3> ... < / h3>
  • < h4> ... < / h4>
  • < h5> ... < / h5>
  • < h6> ... < / h6>
